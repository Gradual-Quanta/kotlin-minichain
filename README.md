# sdkman
## installation sdkman

 $ curl -s "https://get.sdkman.io" | bash

## usage sdkman
  sdk list <package>
  sdk uninstall <package> x.y.z
  sdk install <package>
  sdk use gradle <package> x.y.z
  
## cleanup cache archives
  sdk flush archives
